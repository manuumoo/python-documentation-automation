Python Documentation Automation

This is a beginner-friendly Python automation project that generates a structured process documentation report from CSV data.

The goal of this project is to demonstrate practical skills in Python, automation, data handling, technical documentation, and process improvement.

Project Overview

This project reads process information from a CSV file and automatically generates a Markdown report containing:

* Process name
* Owner
* Status
* Last update
* Priority
* Notes
* Summary
* Recommendations

Technologies Used

* Python
* CSV
* Markdown
* File handling
* Basic automation
* Technical documentation

How It Works

1. The script reads process data from data/processes.csv.
2. It counts the total number of processes reviewed.
3. It identifies high-priority processes.
4. It generates a structured Markdown report.
5. The final report is saved in the output/ folder.
